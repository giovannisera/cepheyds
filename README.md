# Hubble Constant Estimation via Cepheid Variables

This project implements a numerical pipeline in **Fortran 90** to estimate the **Hubble Constant ($H_0$)** and the age of the Universe. The analysis is based on the light curves of 30 Cepheid stars observed in 10 different galaxies, applying advanced numerical methods for cosmic distance determination.

The project also includes a cosmological analysis based on the $\Lambda$-CDM model, comparing the obtained results with data from the Planck satellite (2018).

## ðŸ”­ Project Overview

The goal is to calculate the expansion rate of the Universe ($H_0$) using the "cosmic distance ladder" method, following these steps:
1.  **Calibration:** Fitting the Period-Luminosity relationship (Leavitt's Law).
2.  **Light Curve Analysis:** Determination of the pulsation period and mean magnitude via Spline interpolation.
3.  **Distances:** Calculation of distance moduli for 10 target galaxies.
4.  **Cosmology:** Estimation of $H_0$ and analysis of density parameters ($\Omega_m, \Omega_\Lambda$).

[cite_start]**Result:** $H_0 = 76.54 \pm 1.22 \text{ km s}^{-1} \text{ Mpc}^{-1}$ [cite: 297]

The code is modular and implements the following algorithms from scratch (no external math libraries used):

* [cite_start]**Interpolation:** Cubic Spline for light curve reconstruction[cite: 355].

* [cite_start]**Integration:** Gaussian Quadrature (4-point Gauss-Legendre) for calculating the age of the Universe[cite: 386].
* [cite_start]**Statistics:** Weighted mean with analytical error propagation[cite: 327].
